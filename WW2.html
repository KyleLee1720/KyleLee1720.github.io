Written Response

The purpose of the code we made was to make an educational game. Some problems me and my partner ran into during the making of this game is that the original code was not working the way we wanted it to in that it didn’t randomize the question which was a requirement. So I decided to make a new code that made it easier and more compatible to the random.shuffle code and that solved the problem of randomization of the questions. Then we ran into the problem of how are code wouldn’t allow to have four answers but there was an easy solution what we did is just have another answer input capability then we added the fourth answer. If we had more time I would have tried to randomize the answers so the question would be randomized and everytime you got the same question you can’t just remember were the answers are.
